1. ### 项目概况
- #### 名称
   *Exitless Day*
- #### 类型  
基于 *HTML5+CSS3+Javascript* 平台和 *toml* 技术的“文字冒险闯关复合型”游戏
- #### 企划原型与背景  
  *本项目剧情为原创故事*  
  
  故事发生在一个循环的梦境世界中。主角“猫猫”在一次事故后陷入昏迷，在梦境/精神世界中被困在一段不断重复的“高一的第一天”。每一天都与前一天极为相似，但逐渐出现细微的异常，语言和人物身份开始扭曲，现实与虚幻的界限模糊。
  表面上，这个世界温馨、美好：有慈爱的“妈妈”、亲密的朋友（凌华、千鹤、团团等），每天都有愉快的校园生活与温暖的家庭时光。
  但随着时间推移，猫猫逐渐发现一切在重复、扭曲，世界也开始崩坏。她需要在幻境与现实之间做出选择，最终走向不同结局。

3. ### 游戏设定
- #### 项目角色介绍
  *主要操作角色：*    
    - **猫猫**（默认名字，玩家可更改为个性化昵称）：玩家所操控的唯一角色。所有选择、对话、小游戏操作都由玩家以猫猫的身份进行。  

  *主要角色：*  
    - **猫猫**（主角/玩家操控角色）：  
  一名因车祸而昏迷的高中生。在精神世界中失忆，被困在“高一第一天”的无限循环。内心有对幸福生活的渴望，也有现实焦虑与矛盾。通过她的视角，玩家逐步揭开故事真相。 

    - **妈妈**  
  在梦境中最稳定出现的角色。起初表现为温柔的母亲，唤猫猫起床、准备丰盛早餐。随着循环推进，语言逐渐扭曲，身份可疑（有时口型透露“我一直在等你”）。  
  真实身份：现实中的母亲，一直守护在病床旁，但梦境中的“妈妈”也象征猫猫对家庭温暖与束缚的矛盾心理。  

    - **凌华 / 千鹤 / 团团 / 瑞吉尔等** 
  梦境中出现的不同同学身份。每次都自称是猫猫的朋友，陪伴她度过校园的一天。实际上对应现实中猫猫的高中同学，象征她对友情的留恋。随着循环推进，身份不断错乱、名字替换，逐渐揭示梦境的不真实。

- **故事纲要**  
    猫猫在一个陌生又熟悉的“循环日”醒来：妈妈温柔唤她起床、去上学、和朋友度过愉快的一天、回家吃晚饭、夜晚再次入睡……
    然而，随着循环次数增加，猫猫发现“同一天”在反复发生，并逐渐崩坏：人物对话开始混乱、母亲身份可疑、朋友名字反复错乱。
    在不同的选择与小游戏结果下，猫猫将迎来多种结局……  
- 游戏内容   

  - 第一天：mini game 闯关  
  玩家进入游戏，经历第一天的剧情之后，在夜晚进行一个小游戏闯关。  
    - 胜利：玩家获得一些记忆
    - 失败：继续前进  
    （后续规则相同不再赘述）
  - 第二天：发现重复  
  玩家注意到时间/事件发生了重复。在夜晚仍然进行小游戏闯关。  
  继续前进 → 根据前两天的小游戏输赢情况，第三天出现分支  
  玩家意识到世界开始出现异常。

  - 第三天：世界崩坏的征兆  
  在小游戏中至少获得过一次胜利的玩家进入场景1，其中出现人物动作和语言异常，否则进入崩坏更严重的场景2，出现乱码现象，整体世界开始崩塌。  
  玩家必须进行全新的“小游戏1”考验，本轮小游戏设置两次挑战机会，第二次将挑战“小游戏1 plus”：  
    - 两次胜利胜利 → 进入第四天场景A  
    - 一次胜利一次失败 → 进入第四天场景B
    - 两次失败 → 直接进入第五天 
      
  - 第四天：线索与对抗  
  进入更深层的剧情。有两条主要路径：  
    - 场景A:小游戏2闯关 → 发现真相不对劲  
      - 胜利 → 进入 END 1  
      - 失败 → END 5（失败结局）  
    - 场景B：Q&A 环节（两次抉择）  
        - 抉择一：爱与不爱  
        - 抉择二：相信与不相信  
          - 选择相信的玩家进入第五天  
          - 选择爱但不相信的玩家进入 END 7  
          - 选择不爱但相信的玩家进入 END 6  
  - 第五天：最终抉择  
  如果玩家走到这里，会触发小游戏3  
    - 胜利：进入 END 3
    - 失败：再次进入抉择  
      - 想 -> END 2  
      - 不想 -> END 4

  - 结局设计（ENDings）

    - END 1：我们所厌恶又热爱的生活
    - END 2: 无法逃离的一天
    - END 3：错误的现实
    - END 4：美梦
    - END 5：疯狂幻想家
    - END 6：突如其来的崩坏
    - END 7：物是人非
